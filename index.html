EPICS:
Experimental Physics and Industrial Control System

  • Wire protocol called "Channel Access"
  • Support libraries for distributed hardware control
  • Reference implementation is C/C++
    • 180K lines of code
    • Runs on everything

EPICS:
Experimental Physics and Industrial Control System

  • Started in the 1980s (copyrights of 1986 in the codebase) at Los Alamos
  • Used to control large-scale experimental facilities
    • National Synchrotron Light Source II
    • LIGO (gravity waves!)
    • telescopes, nuclear reactors, accelerators, industry

Sans I/O — sans-io.readthedocs.io

  • Separate implementation of network protocol rules from pushing bits around
  • Have a core state machine that enforces the rules
  • You hand it bits from the wire and it gives you back nice python objects that mean something
  • You hand it nice python objects and hands you back the bits to push over the wire
  • Cory Benfield, Brett Cannon, Nathaniel Smith

Caproto = CA + proto

  • Pure python (3.6+) implementation of CA protocol
  • Have server implementations for
    {asyncio, curio, trio}
  • Have client implementations for {threads, sync}
  • Runs on:
    • Your desktop
    • Your laptop
    • Your raspberry pi
    • Your phone
    • Scales down to hobbyist-scale projects
